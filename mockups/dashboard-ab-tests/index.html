IBR mockup experiment

Dashboard A/B/C scans

Three static dashboards designed to exercise IBR hierarchy, geometry, interaction-state, data-viz, and design-reference scans without a build step.

Variant A

Compact Operations

Dense SaaS dashboard with KPIs, table-first work queue, grouped metrics, and text-color status.

Variant B

Evidence Workbench

Glass workspace with multi-pane evidence review, inspector hierarchy, and action-state coverage.

Variant C

Data Narrative

Dark-to-light analytical dashboard with decision-first charts, bento hierarchy, and source lines.
